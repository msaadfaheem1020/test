# N-Body Simulation

## Setup

Clone the repository:
```bash
git clone https://github.com/humblepenguinn/CAAL-S26-Project-2.0-Gpa-Club --recursive
```

Then run the project using:
```bash
make run # this will build and run the project using cmake
```

To use the `mmap` simulation type, open a seperate terminal window and run:
```bash
make riscv-docker # your going to need docker installed
```
